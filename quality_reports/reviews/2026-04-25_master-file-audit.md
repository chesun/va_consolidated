**Goal:** verify whether `do_all.do` (cde_va_project_fork) and `master.do` (caschls) reference every load-bearing do-file, or whether some only get run by hand.

**Method:** for each repo, list every `.do` file under `do_files/` (fork) or `do/` (caschls), strip extensions, and compare against the set of paths referenced by `do file ...` lines inside the master file. Surface the gap.

**Scope:** post-archival state in cde_va_project_fork (commits c7867e4 + 85a97e7); current state of caschls.
